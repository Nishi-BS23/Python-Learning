## Introduction to Control Flow

### What is Control Flow in Programming?

Control flow refers to the order in which statements are executed in a program. By default, Python code executes line by line, from top to bottom. Control flow statements allow you to change this order, making decisions and repeating actions based on conditions.

Think of control flow as the traffic rules for your program - it determines which path your code takes and when to loop back or skip ahead.

### Importance of Controlling Program Logic

Control flow is essential because:

- **Decision Making**: Allows programs to respond differently based on input or conditions
- **Repetition**: Enables tasks to be repeated without writing the same code multiple times
- **Efficiency**: Helps write cleaner, more organized code
- **Flexibility**: Makes programs dynamic and interactive
- **Problem Solving**: Most real-world problems require conditional logic and loops

Without control flow, programs would be linear and unable to make decisions or handle complex scenarios.

## Conditional Statements

Conditional statements allow your program to make decisions based on conditions. They execute different code blocks depending on whether a condition is True or False.

### if Statement

The `if` statement executes a block of code only if a condition is true.

**Syntax**:
```python
if condition:
    # code to execute if condition is True
```

**Example**:
```python
age = 18

if age >= 18:
    print("You are eligible to vote!")
```

**Output**:
```
You are eligible to vote!
```

### else Statement

The `else` statement executes a block of code when the `if` condition is False.

**Syntax**:
```python
if condition:
    # code if condition is True
else:
    # code if condition is False
```

**Example**:
```python
age = 15

if age >= 18:
    print("You are eligible to vote!")
else:
    print("You are not old enough to vote yet.")
```

**Output**:
```
You are not old enough to vote yet.
```

### elif (Else If)

The `elif` statement allows you to check multiple conditions. It's short for "else if" and is used after an `if` statement.

**Syntax**:
```python
if condition1:
    # code if condition1 is True
elif condition2:
    # code if condition1 is False and condition2 is True
elif condition3:
    # code if condition1 and condition2 are False, condition3 is True
else:
    # code if all conditions are False
```

**Example**:
```python
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")
```

**Output**:
```
Grade: B
```

### Nested Conditional Statements

You can place conditional statements inside other conditional statements. This is called nesting.

**Example**:
```python
age = 25
has_license = True

if age >= 18:
    print("You are old enough to drive.")
    if has_license:
        print("You have a license. You can drive!")
    else:
        print("You need to get a license first.")
else:
    print("You are too young to drive.")
```

**Output**:
```
You are old enough to drive.
You have a license. You can drive!
```

## Boolean Expressions

Boolean expressions are expressions that evaluate to either `True` or `False`. They are the foundation of conditional statements.

### True/False Values

In Python, boolean values are `True` and `False` (note the capitalization).

```python
print(True)   # Output: True
print(False)  # Output: False

# Boolean expressions
print(5 > 3)    # Output: True
print(5 < 3)    # Output: False
print(5 == 5)   # Output: True
print(5 != 3)   # Output: True
```

### Using Logical Operators in Conditions

Logical operators allow you to combine multiple boolean expressions.

#### and Operator

Returns `True` only if both conditions are `True`.

```python
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive.")
else:
    print("You cannot drive.")

# Output: You can drive.
```

#### or Operator

Returns `True` if at least one condition is `True`.

```python
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")
else:
    print("It's a weekday.")

# Output: It's the weekend!
```

#### not Operator

Reverses the boolean value (True becomes False, False becomes True).

```python
is_raining = False

if not is_raining:
    print("Let's go outside!")
else:
    print("Stay inside.")

# Output: Let's go outside!
```

## Loops

Loops allow you to execute a block of code multiple times. Python has two main types of loops: `for` loops and `while` loops.

### For Loops

A `for` loop is used to iterate over a sequence (like a list, string, or range) and execute a block of code for each item.

#### Syntax and Example

**Syntax**:
```python
for variable in sequence:
    # code to execute for each item
```

**Basic Example**:
```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
```

**Output**:
```
apple
banana
cherry
```

#### Iterating Over a List

```python
numbers = [10, 20, 30, 40, 50]

for num in numbers:
    print(num)
```

**Output**:
```
10
20
30
40
50
```

#### Iterating Over a Range

The `range()` function generates a sequence of numbers.

```python
# range(5) generates 0, 1, 2, 3, 4
for i in range(5):
    print(i)
```

**Output**:
```
0
1
2
3
4
```

**Range with start and stop**:
```python
# range(1, 6) generates 1, 2, 3, 4, 5
for i in range(1, 6):
    print(i)
```

**Range with step**:
```python
# range(0, 10, 2) generates 0, 2, 4, 6, 8
for i in range(0, 10, 2):
    print(i)
```

**Output**:
```
0
2
4
6
8
```

#### Iterating Over a String

```python
message = "Python"

for char in message:
    print(char)
```

**Output**:
```
P
y
t
h
o
n
```

### While Loops

A `while` loop repeats as long as a condition is true. It's useful when you don't know in advance how many times you need to loop.

#### Syntax and Example

**Syntax**:
```python
while condition:
    # code to execute while condition is True
```

**Basic Example**:
```python
count = 0

while count < 5:
    print(count)
    count += 1  # Important: update the condition variable!
```

**Output**:
```
0
1
2
3
4
```

#### Loop That Runs Until a Condition is Met

```python
# Keep asking for input until user enters 'quit'
user_input = ""

while user_input != "quit":
    user_input = input("Enter something (or 'quit' to exit): ")
    print("You entered:", user_input)
```

**Example Run**:
```
Enter something (or 'quit' to exit): hello
You entered: hello
Enter something (or 'quit' to exit): python
You entered: python
Enter something (or 'quit' to exit): quit
You entered: quit
```

**Important**: Always make sure the loop condition will eventually become False, or you'll create an infinite loop!

### Nested Loops

You can place loops inside other loops. This is called nesting.

```python
# Example: Printing a pattern
for i in range(3):
    for j in range(3):
        print(f"i={i}, j={j}")
    print("---")
```

**Output**:
```
i=0, j=0
i=0, j=1
i=0, j=2
---
i=1, j=0
i=1, j=1
i=1, j=2
---
i=2, j=0
i=2, j=1
i=2, j=2
---
```

**Example: Multiplication Table**:
```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i * j}", end="  ")
    print()  # New line after each row
```

**Output**:
```

2 x 1 = 2  2 x 2 = 4  2 x 3 = 6  
3 x 1 = 3  3 x 2 = 6  3 x 3 = 9  
```

## Loop Control Statements

Python provides statements to control the flow within loops: `break`, `continue`, and `pass`.

### break – Exit a Loop

The `break` statement immediately exits the loop, regardless of the condition.

```python
for i in range(10):
    if i == 5:
        break  # Exit the loop when i equals 5
    print(i)
```

**Output**:
```
0
1
2
3
4
```

**Example with while loop**:
```python
count = 0

while True:  # Infinite loop
    print(count)
    count += 1
    if count == 5:
        break  # Exit the loop
```

**Output**:
```
0
1
2
3
4
```

### continue – Skip Current Iteration

The `continue` statement skips the rest of the current iteration and moves to the next one.

```python
for i in range(5):
    if i == 2:
        continue  # Skip when i equals 2
    print(i)
```

**Output**:
```
0
1
3
4
```

**Example: Print only odd numbers**:
```python
for i in range(10):
    if i % 2 == 0:
        continue  # Skip even numbers
    print(i)
```

**Output**:
```
1
3
5
7
9
```

### pass – Placeholder Statement

The `pass` statement does nothing. It's used as a placeholder when you need a statement syntactically but don't want to execute any code.

```python
for i in range(5):
    if i == 2:
        pass  # Do nothing when i equals 2
    else:
        print(i)
```

**Output**:
```
0
1
3
4
```

**Common use case**: Empty function or class definition
```python
def my_function():
    pass  # TODO: implement this function later

class MyClass:
    pass  # TODO: add methods later
```

## Exercises

### Exercise 1: Check if a Number is Prime

Write a program that checks if a number entered by the user is prime.

**Hint**: A prime number is only divisible by 1 and itself.

```python
# Solution
number = int(input("Enter a number: "))

if number < 2:
    print(f"{number} is not prime")
else:
    is_prime = True
    for i in range(2, int(number ** 0.5) + 1):
        if number % i == 0:
            is_prime = False
            break
    
    if is_prime:
        print(f"{number} is prime")
    else:
        print(f"{number} is not prime")
```

### Common Mistakes to Avoid

- **Forgetting the colon** `:` after if, elif, else, for, while
- **Incorrect indentation** - Python uses indentation to define blocks
- **Infinite loops** - Always ensure while loop conditions can become False
- **Using `=` instead of `==`** - `=` is assignment, `==` is comparison
- **Forgetting to update loop variables** in while loops
- **Off-by-one errors** in range() - remember range(5) gives 0,1,2,3,4
