# Python Basics - Module 1

## Introduction to Python

### What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python emphasizes code readability with its notable use of significant whitespace.

### Why is Python Popular?

- **Easy to Learn**: Python has a simple syntax that resembles natural language, making it perfect for beginners
- **Versatile**: It can be used for almost any type of programming task
- **Large Community**: A vast community of developers contributes to libraries and provides support
- **Cross-Platform**: Python runs on Windows, Mac, Linux, and other operating systems
- **Extensive Libraries**: Thousands of libraries are available for various tasks

### Applications of Python

Python is used in many fields:

1. **Web Development** - Building websites and web applications (Django, Flask, FastAPI)
2. **Data Science** - Analyzing and visualizing data (Pandas, NumPy, Matplotlib)
3. **Machine Learning** - Creating AI models (TensorFlow, PyTorch, Scikit-learn)
4. **Automation** - Automating repetitive tasks
5. **Scripting** - Writing small utility scripts
6. **Game Development** - Creating games (Pygame)
7. **Desktop Applications** - Building GUI applications (Tkinter, PyQt)

### Python Versions

- **Python 2.x**: The older version (no longer supported since 2020)
- **Python 3.x**: The current and recommended version

We will focus on **Python 3** throughout this course. Python 3 includes many improvements and fixes over Python 2.

## Installing Python and Setup

### Installing Python on Different Operating Systems

#### Windows

1. Visit the official Python website: [python.org](https://www.python.org)
2. Click on "Downloads" and download the latest Python 3 installer for Windows
3. Run the installer
4. **Important**: Check the box "Add Python to PATH" during installation
5. Click "Install Now"

#### Mac

```bash

```

**Using Official Installer**:
1. Visit [python.org](https://www.python.org)
2. Download the macOS installer
3. Run the installer and follow the prompts

#### Linux

Most Linux distributions come with Python pre-installed. To install or update:

**Ubuntu/Debian**:
```bash
sudo apt update
sudo apt install python3
```

**Fedora**:
```bash
sudo dnf install python3
```

An IDE helps you write, run, and debug code efficiently.

1. **VS Code** (Visual Studio Code)
   - Free, lightweight, and highly customizable

2. **PyCharm**

3. **Jupyter Notebook**

### How to Run a Python Script

#### From Terminal/Command Prompt

1. Create a file named `hello.py` with your code
2. Open terminal/command prompt
3. Navigate to the file's directory
4. Run: `python3 hello.py` (or `python hello.py` on Windows)

#### From IDE

- **VS Code**: Right-click the file and select "Run Python File in Terminal" or press F5
- **PyCharm**: Click the green play button or right-click and select "Run"
- **Jupyter**: Click on a code cell and press Shift+Enter

## Your First Python Program

### Hello, World! Example

```python
print("Hello, World!")
```

**Output**:
```
Hello, World!
```

### Explanation of the print() Function

- `print()` is a built-in function that outputs text to the screen
- The text you want to print must be inside parentheses
- Text must be enclosed in quotes (single `'` or double `"`)
- `print()` can also print numbers, variables, and expressions

```python
print("Hello, World!")      # Prints text
print(42)                   # Prints a number
print(3 + 5)                # Prints the result of the calculation
```

**Output**:
```
Hello, World!
42
8
```

## Variables and Data Types

### What are Variables?

A variable is like a container that stores data. You can think of it as a labeled box where you keep information.

```python
name = "Alice"      # A variable named 'name' storing "Alice"
age = 25            # A variable named 'age' storing 25
```

### Basic Data Types

Python has several built-in data types:

#### 1. Integer (int)
Whole numbers without decimal points.

```python
x = 10
y = -5
z = 0
```

#### 2. Float (float)
Numbers with decimal points.

```python
price = 19.99
temperature = -3.5
pi = 3.14159
```

#### 3. String (str)
Text enclosed in quotes.

```python
message = "Hello, World!"
greeting = 'Good morning'
empty_string = ""
```

#### 4. Boolean (bool)
Represents True or False values.

```python
is_student = True
has_license = False
is_raining = True
```

### Examples of Assigning and Printing Variables

```python
# Assigning values to variables
name = "John"
age = 30
height = 5.9
is_employed = True

# Printing variables
print(name)           # Output: John
print(age)            # Output: 30
print(height)         # Output: 5.9
print(is_employed)    # Output: True

# Printing with messages
print("Name:", name)
print("Age:", age)
```

### Type Conversion

Sometimes you need to convert one data type to another.

```python
# Converting to integer
x = int("42")        # Converts string "42" to integer 42
y = int(3.9)         # Converts float 3.9 to integer 3 (truncates)

# Converting to float
a = float("3.14")    # Converts string "3.14" to float 3.14
b = float(10)        # Converts integer 10 to float 10.0

# Converting to string
s1 = str(100)        # Converts integer 100 to string "100"
s2 = str(3.14)       # Converts float 3.14 to string "3.14"

print(x)             # Output: 42
print(type(x))       # Output: <class 'int'>
print(s1)            # Output: 100
print(type(s1))      # Output: <class 'str'>
```

## Basic Operators

### Arithmetic Operators

Used to perform mathematical operations.

| Operator | Name | Example | Result |
|----------|------|---------|--------|
| `+` | Addition | `5 + 3` | 8 |
| `-` | Subtraction | `5 - 3` | 2 |
| `*` | Multiplication | `5 * 3` | 15 |
| `/` | Division | `5 / 3` | 1.666... |
| `%` | Modulus (remainder) | `5 % 3` | 2 |
| `**` | Exponentiation | `5 ** 3` | 125 |
| `//` | Floor Division | `5 // 3` | 1 |

```python
print(5 + 3)     # Output: 8
print(10 - 4)    # Output: 6
print(6 * 7)     # Output: 42
print(15 / 3)    # Output: 5.0
print(17 % 5)    # Output: 2 (remainder of 17/5)
print(2 ** 8)    # Output: 256 (2 to the power of 8)
print(17 // 5)   # Output: 3 (floor division)
```

### Comparison Operators

Used to compare values. They return `True` or `False`.

| Operator | Name | Example | Result |
|----------|------|---------|--------|
| `==` | Equal to | `5 == 5` | True |
| `!=` | Not equal to | `5 != 3` | True |
| `>` | Greater than | `5 > 3` | True |
| `<` | Less than | `3 < 5` | True |
| `>=` | Greater than or equal | `5 >= 5` | True |
| `<=` | Less than or equal | `3 <= 5` | True |

```python
print(5 == 5)    # Output: True
print(5 == 3)    # Output: False
print(5 != 3)    # Output: True
print(5 > 3)     # Output: True
print(5 < 3)     # Output: False
print(5 >= 5)    # Output: True
print(3 <= 5)    # Output: True
```

### Logical Operators

Used to combine conditional statements.

| Operator | Description | Example | Result |
|----------|-------------|---------|--------|
| `and` | Returns True if both conditions are True | `True and True` | True |
| `or` | Returns True if at least one condition is True | `True or False` | True |
| `not` | Reverses the result | `not True` | False |

```python
# and operator
print(True and True)    # Output: True
print(True and False)   # Output: False
print(False and False)  # Output: False

# or operator
print(True or True)     # Output: True
print(True or False)    # Output: True
print(False or False)   # Output: False

# not operator
print(not True)         # Output: False
print(not False)        # Output: True

# Practical examples
age = 25
print(age >= 18 and age <= 30)    # Output: True
print(age < 18 or age > 65)       # Output: False
print(not (age < 18))             # Output: True
```

## Input and Output

### Using input() to Get User Input

The `input()` function allows you to get input from the user. It always returns a string.

```python
name = input("Enter your name: ")
print("Hello, " + name + "!")
```

**Example Run**:
```
Enter your name: Alice
Hello, Alice!
```

### Converting Input to Appropriate Data Types

Since `input()` always returns a string, you often need to convert it to other types.

```python
# Getting a number as input
age = input("Enter your age: ")
age = int(age)          # Convert string to integer
print("Next year you will be", age + 1)

# Or do it in one line
age = int(input("Enter your age: "))
print("Next year you will be", age + 1)
```

### Example: Taking Two Numbers and Printing Their Sum

```python
# Get first number
num1 = int(input("Enter first number: "))

# Get second number
num2 = int(input("Enter second number: "))

# Calculate sum
sum = num1 + num2

# Print result
print("The sum is:", sum)
```

**Example Run**:
```
Enter first number: 10
Enter second number: 20
The sum is: 30
```

### More Input/Output Examples

```python
# String input
city = input("Enter your city: ")
print("You live in", city)

# Float input
price = float(input("Enter price: "))
tax = price * 0.1
print("Tax (10%):", tax)

# Multiple inputs
first_name = input("First name: ")
last_name = input("Last name: ")
full_name = first_name + " " + last_name
print("Full name:", full_name)
```

## Comments and Docstrings

### Single-Line Comments

Comments are notes in your code that are ignored by Python. They help explain your code.

```python
# This is a single-line comment
print("Hello")  # This comment is after code

# Calculate the area
area = length * width
```

### Multi-Line Comments / Docstrings

For longer comments or documentation, use triple quotes.

```python
"""
This is a multi-line comment.
It can span multiple lines.
Python ignores everything between the triple quotes.
"""

print("Hello")

# Using triple quotes for docstrings (function documentation)
def greet(name):
    """
    This function greets the person with the given name.
    
    Parameters:
    name (str): The name of the person to greet
    
    Returns:
    str: A greeting message
    """
    return "Hello, " + name
```

### Importance of Commenting Code

- **Documentation**: Helps others understand your code
- **Maintenance**: Makes it easier to fix bugs later
- **Collaboration**: Essential when working in teams
- **Reminder**: Helps you remember why you wrote code a certain way

**Best Practices**:
- Write clear, concise comments
- Comment **why**, not **what** (the code shows what)
- Keep comments up to date
- Don't over-comment obvious code

**Good Comment Example**:
```python
# Calculate discount for loyal customers (10% off)
discount = total * 0.10
```

**Bad Comment Example**:
```python
# Multiply total by 0.10
discount = total * 0.10
```

## Simple Exercises

### Exercise 1: Print Your Name and Age

Write a program that prints your name and age.

```python
# Solution
name = "Your Name"
age = 25

print("Name:", name)
print("Age:", age)
```

### Exercise 2: Calculate Area of a Rectangle

Write a program that calculates and prints the area of a rectangle.

**Formula**: Area = length × width

```python
# Solution
length = 10
width = 5

area = length * width
print("Length:", length)
print("Width:", width)
print("Area of rectangle:", area)
```

**Challenge**: Modify the program to take length and width as user input.

```python
# Challenge Solution
length = float(input("Enter length: "))
width = float(input("Enter width: "))

area = length * width
print("Area of rectangle:", area)
```

### Exercise 3: Check if a Number is Even or Odd

Write a program that takes a number from the user and checks if it's even or odd.

**Hint**: Use the modulus operator `%`. If `number % 2 == 0`, it's even.

```python
# Solution
number = int(input("Enter a number: "))

if number % 2 == 0:
    print(number, "is even")
else:
    print(number, "is odd")
```

### Exercise 4: Swap Two Numbers Using Variables

Write a program that swaps the values of two variables.

```python
# Solution
a = 5
b = 10

print("Before swapping:")
print("a =", a)
print("b =", b)

# Swap using a temporary variable
temp = a
a = b
b = temp

print("\nAfter swapping:")
print("a =", a)
print("b =", b)
```

**Alternative Method** (Python-specific):
```python
# Python allows swapping in one line
a = 5
b = 10

a, b = b, a  # Swap in one line!

print("a =", a)  # Output: 10
print("b =", b)  # Output: 5
```
