## Introduction to Functions

### What are Functions?

A function is a reusable block of code that performs a specific task. Think of a function as a recipe - you write the instructions once, and you can use it whenever you need to prepare that dish.

In programming, functions allow you to:
- Group related code together
- Give it a meaningful name
- Call it multiple times from different parts of your program

### Why Functions are Useful

Functions are fundamental to good programming for several reasons:

- **Code Reusability**: Write once, use many times. No need to copy-paste the same code.
- **Readability**: Functions break complex problems into smaller, manageable pieces
- **Organization**: Group related functionality together
- **Maintenance**: Fix a bug in one place, and it's fixed everywhere the function is used
- **Testing**: Easier to test small, isolated functions than large programs
- **Abstraction**: Hide complex implementation details behind a simple interface

**Example Without Function**:
```python
# Calculating area of different rectangles
length1, width1 = 5, 3
area1 = length1 * width1
print(area1)

length2, width2 = 10, 4
area2 = length2 * width2
print(area2)

length3, width3 = 7, 2
area3 = length3 * width3
print(area3)
```

**Example With Function**:
```python
def calculate_area(length, width):
    return length * width

print(calculate_area(5, 3))
print(calculate_area(10, 4))
print(calculate_area(7, 2))
```

Much cleaner and easier to maintain!

## Defining Functions

### Syntax Using def Keyword

Functions are defined using the `def` keyword followed by the function name, parentheses, and a colon.

**Basic Syntax**:
```python
def function_name(parameters):
    # Function body
    # Code to execute
    return result  # Optional
```

### Function Name and Parameters

- **Function Name**: Should be descriptive and follow Python naming conventions (lowercase with underscores)
- **Parameters**: Variables that receive values when the function is called (optional)

```python
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Output: Hello, Alice!
greet("Bob")    # Output: Hello, Bob!
```

### Returning Values Using return

The `return` statement sends a value back to the caller. A function can return one value, multiple values, or nothing.

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8
```

**Returning Multiple Values**:
```python
def get_name_and_age():
    return "Alice", 25

name, age = get_name_and_age()
print(name)  # Output: Alice
print(age)   # Output: 25
```

**No Return Statement** (returns None):
```python
def say_hello():
    print("Hello!")

result = say_hello()
print(result)  # Output: None
```

### Example: Simple Function to Add Two Numbers

```python
def add_numbers(num1, num2):
    """Add two numbers and return the result."""
    result = num1 + num2
    return result

# Calling the function
sum_result = add_numbers(10, 20)
print(f"The sum is: {sum_result}")  # Output: The sum is: 30

# Using it directly
print(add_numbers(5, 7))  # Output: 12
```

## Function Parameters

Parameters are variables listed in the function definition. Arguments are the actual values passed when calling the function.

### Positional Arguments

Arguments are matched to parameters based on their position.

```python
def greet(name, age):
    print(f"Hello {name}, you are {age} years old.")

greet("Alice", 25)  # Output: Hello Alice, you are 25 years old.
greet("Bob", 30)    # Output: Hello Bob, you are 30 years old.
```

The first argument goes to the first parameter, second to second, and so on.

### Keyword Arguments

You can specify arguments by parameter name, regardless of position.

```python
def greet(name, age):
    print(f"Hello {name}, you are {age} years old.")

# Using keyword arguments
greet(age=25, name="Alice")  # Output: Hello Alice, you are 25 years old.

# Mixing positional and keyword (keyword arguments must come last)
greet("Bob", age=30)  # Output: Hello Bob, you are 30 years old.
```

### Default Parameter Values

You can provide default values for parameters. If an argument is not provided, the default is used.

```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")                    # Output: Hello, Alice!
greet("Bob", "Hi")                # Output: Hi, Bob!
greet("Charlie", greeting="Hey")  # Output: Hey, Charlie!
```

**Example with Multiple Defaults**:
```python
def calculate_price(price, tax=0.1, discount=0):
    final_price = price * (1 + tax) - discount
    return final_price

print(calculate_price(100))              # Output: 110.0
print(calculate_price(100, 0.15))        # Output: 115.0
print(calculate_price(100, 0.1, 10))     # Output: 100.0
```

### Variable-Length Arguments (Optional)

Python allows functions to accept a variable number of arguments using `*args` and `**kwargs`.

#### *args - Variable Positional Arguments

Collects extra positional arguments into a tuple.

```python
def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3))        # Output: 6
print(sum_all(1, 2, 3, 4, 5))  # Output: 15
print(sum_all())               # Output: 0
```

#### **kwargs - Variable Keyword Arguments

Collects extra keyword arguments into a dictionary.

```python
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="New York")
# Output:
# name: Alice
# age: 25
# city: New York
```

**Combining Both**:
```python
def display_info(name, *args, **kwargs):
    print(f"Name: {name}")
    print("Additional args:", args)
    print("Keyword args:", kwargs)

display_info("Alice", 1, 2, 3, age=25, city="NY")
# Output:
# Name: Alice
# Additional args: (1, 2, 3)
# Keyword args: {'age': 25, 'city': 'NY'}
```

## Calling Functions

### How to Call Functions

To use a function, you call it by its name followed by parentheses containing the arguments.

```python
def say_hello(name):
    return f"Hello, {name}!"

# Calling the function
message = say_hello("Alice")
print(message)  # Output: Hello, Alice!
```

### Using Returned Values

Functions can return values that you can use in expressions, store in variables, or pass to other functions.

```python
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

# Using return value in expression
result = add(5, 3) * 2
print(result)  # Output: 16

# Passing return value to another function
sum_result = add(10, 20)
final = multiply(sum_result, 2)
print(final)  # Output: 60

# Using directly in print
print(multiply(add(5, 5), 3))  # Output: 30
```

### Example: Function to Calculate Factorial

The factorial of a number n (written as n!) is the product of all positive integers from 1 to n.

```python
def factorial(n):
    """Calculate the factorial of a number."""
    if n < 0:
        return "Factorial is not defined for negative numbers"
    if n == 0 or n == 1:
        return 1
    
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result

# Testing the function
print(factorial(5))   # Output: 120 (5 × 4 × 3 × 2 × 1)
print(factorial(0))   # Output: 1
print(factorial(1))   # Output: 1
print(factorial(10))  # Output: 3628800
print(factorial(-3))  # Output: Factorial is not defined for negative numbers
```

**Recursive Version** (functions calling themselves):
```python
def factorial_recursive(n):
    """Calculate factorial using recursion."""
    if n <= 1:
        return 1
    return n * factorial_recursive(n - 1)

print(factorial_recursive(5))  # Output: 120
```

## Lambda Functions

### Introduction to Anonymous Functions

Lambda functions are small, anonymous functions defined using the `lambda` keyword. They can have any number of arguments but only one expression.

**Why Use Lambda Functions**:
- For short, simple operations
- When you need a function for a short time
- As arguments to higher-order functions (like map, filter, sort)

### Syntax: lambda arguments: expression

```python
lambda arguments: expression
```

- `lambda`: Keyword to define the function
- `arguments`: Comma-separated parameters (like regular functions)
- `expression`: Single expression that is evaluated and returned

### Simple Examples

#### Squaring a Number

```python
# Regular function
def square(x):
    return x ** 2

print(square(5))  # Output: 25

# Lambda function
square_lambda = lambda x: x ** 2
print(square_lambda(5))  # Output: 25
```

#### Sum of Two Numbers

```python
# Regular function
def add(a, b):
    return a + b

print(add(3, 5))  # Output: 8

# Lambda function
add_lambda = lambda a, b: a + b
print(add_lambda(3, 5))  # Output: 8
```

#### More Lambda Examples

```python
# Check if a number is even
is_even = lambda x: x % 2 == 0
print(is_even(4))   # Output: True
print(is_even(7))   # Output: False

# Get the first character of a string
first_char = lambda s: s[0]
print(first_char("Hello"))  # Output: H

# Calculate area of rectangle
area = lambda l, w: l * w
print(area(5, 3))  # Output: 15
```

#### Lambda with Built-in Functions

```python
# Using with sorted()
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
# Sort by score (second element)
sorted_students = sorted(students, key=lambda x: x[1])
print(sorted_students)
# Output: [('Charlie', 78), ('Alice', 85), ('Bob', 92)]

# Using with map() - square each number
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Output: [1, 4, 9, 16, 25]

# Using with filter() - keep even numbers
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # Output: [2, 4]
```

## Modules

### What is a Module?

A module is a Python file containing Python definitions and statements. It allows you to organize your code into separate files and reuse code across different programs.

Think of modules as:
- Toolboxes containing specific tools (functions, variables, classes)
- Libraries of reusable code
- A way to break large programs into smaller, manageable files

### Why Modules are Important

- **Organization**: Group related functionality together
- **Reusability**: Use the same code in multiple programs
- **Namespace Avoidance**: Prevent naming conflicts between files
- **Maintainability**: Easier to update and fix code in one place
- **Collaboration**: Different team members can work on different modules

Python comes with many built-in modules, and you can create your own.

### Importing Modules

#### Using import

Imports the entire module. You access functions using `module_name.function_name`.

```python
import math

# Using math module functions
print(math.pi)          # Output: 3.141592653589793
print(math.sqrt(16))    # Output: 4.0
print(math.pow(2, 3))   # Output: 8.0
print(math.floor(3.7))  # Output: 3
print(math.ceil(3.2))   # Output: 4
```

#### Using from … import …

Imports specific functions or variables from a module. You can use them directly without the module name.

```python
from math import pi, sqrt

# Can use directly without math. prefix
print(pi)         # Output: 3.141592653589793
print(sqrt(16))   # Output: 4.0
```

#### Importing All (Not Recommended)

```python
from math import *

# Now all math functions are available directly
print(pi)
print(sqrt(25))
```

**Caution**: This is generally discouraged because it can cause naming conflicts and makes code harder to read.

#### Using Alias

You can give a module a shorter name using `as`.

```python
import math as m

print(m.pi)       # Output: 3.141592653589793
print(m.sqrt(16))  # Output: 4.0
```

### Example: Using Python's Built-in math Module

```python
import math

# Constants
print(f"Pi: {math.pi}")
print(f"Euler's number: {math.e}")

# Basic functions
print(f"Square root of 16: {math.sqrt(16)}")

# Conversion between degrees and radians

radians = math.pi / 2
degrees = math.degrees(radians)
print(f"{radians} radians = {degrees}°")
```

## Creating Your Own Module

### How to Create a Python File and Use It as a Module

Creating your own module is simple - just create a Python file with functions, and you can import it in other Python files.

**Steps**:
1. Create a Python file (e.g., `my_module.py`)
2. Write functions and variables in that file
3. Import the module in another Python file using `import`

### Example: Create my_module.py with a Function and Import It

**File: my_module.py**
```python
# my_module.py - A custom module

def greet(name):
    """Greet a person by name."""
    return f"Hello, {name}!"

def calculate_area(length, width):
    """Calculate the area of a rectangle."""
    return length * width

PI = 3.14159

def circle_area(radius):
    """Calculate the area of a circle."""
    return PI * radius ** 2
```

**File: main.py** (or in the same directory)
```python
import my_module

# Using functions from my_module
print(my_module.greet("Alice"))
print(my_module.calculate_area(5, 3))
print(my_module.circle_area(4))
print(my_module.PI)
```

**Alternative: Using from … import**
```python
from my_module import greet, calculate_area, circle_area, PI

# Use functions directly
print(greet("Bob"))
print(calculate_area(10, 4))
print(circle_area(5))
print(PI)
```

**Output**:
```
Hello, Alice!
15
50.26544
3.14159
```

**Important**: The module file must be in the same directory as your main script, or in Python's module search path.

### Example 1: Function to Check if a Number is Prime

```python
def is_prime(n):
    """Check if a number is prime."""
    if n < 2:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    
    # Check odd divisors up to square root of n
    for i in range(3, int(n ** 0.5) + 1, 2):
        if n % i == 0:
            return False
    return True

# Testing the function
test_numbers = [2, 3, 4, 5, 7, 11, 15, 17, 20, 23]

for num in test_numbers:
    if is_prime(num):
        print(f"{num} is prime")
    else:
        print(f"{num} is not prime")
```

def circle_area(radius):

### Example 3: Using random Module to Simulate a Dice Roll

```python
import random

def roll_dice(sides=6):
    """Simulate rolling a dice with given number of sides."""
    return random.randint(1, sides)

def roll_multiple_dice(num_dice, sides=6):
    """Roll multiple dice and return the results."""
    results = []
    for _ in range(num_dice):
        results.append(roll_dice(sides))
    return results

# Single dice roll
print("Rolling a 6-sided dice:")
for i in range(5):
    result = roll_dice()
    print(f"Roll {i+1}: {result}")

print("\nRolling two 6-sided dice:")
for i in range(3):
    results = roll_multiple_dice(2)
    total = sum(results)
    print(f"Roll {i+1}: {results} = {total}")

print("\nRolling a 20-sided dice:")
for i in range(5):
    result = roll_dice(20)
    print(f"Roll {i+1}: {result}")
```

### Exercise 1: Write a Function that Returns the Largest of Three Numbers

Write a function that takes three numbers as arguments and returns the largest one.

```python
# Solution
def largest_of_three(a, b, c):
    """Return the largest of three numbers."""
    if a >= b and a >= c:
        return a
    elif b >= a and b >= c:
        return b
    else:
        return c

# Alternative using built-in max()
def largest_of_three_v2(a, b, c):
    return max(a, b, c)

# Testing
print(largest_of_three(10, 20, 30))  # Output: 30
print(largest_of_three(50, 25, 40))  # Output: 50
print(largest_of_three(5, 5, 5))     # Output: 5

print(largest_of_three_v2(10, 20, 30))  # Output: 30
```

### Exercise 2: Write a Function to Reverse a String

Write a function that takes a string and returns it reversed.

```python

def reverse_string(s):
    """Reverse a string using slicing."""
    return s[::-1]

# Solution 2: Using loop
def reverse_string_v2(s):
    """Reverse a string using a loop."""
    reversed_str = ""
    for char in s:
        reversed_str = char + reversed_str
    return reversed_str

# Solution 3: Using reversed()
def reverse_string_v3(s):
    """Reverse a string using reversed()."""
    return "".join(reversed(s))

# Testing
print(reverse_string("hello"))      # Output: olleh
print(reverse_string_v2("Python"))   # Output: nohtyP
print(reverse_string_v3("World"))    # Output: dlroW
```
